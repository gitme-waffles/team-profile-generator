## Installation Instructions
Requires : 
- `node.js`
- npm 
   - `inquirer`
   - `jest`

Open terminal or git bash and run the following command:

```
npm install
```

 
## Usage 
Once the modules are installed, run the following command in the next line:
```
node index.js
```
